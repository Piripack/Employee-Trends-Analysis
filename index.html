Employee Trends Analysis

💻 Type: Data Analysis
🎯 Role: Data Analyst

Project Overview

The Employee Trends Analysis project involved analyzing employee data across various dimensions including demographics, education, job roles, and satisfaction levels. The dataset includes information about gender, age, department, education, travel patterns, and attrition.

Project Background

The task focused on analyzing:

  • Employee demographics (gender, age bands, marital status)
  • Professional attributes (department, education, job roles)
  • Satisfaction and retention metrics (job satisfaction, attrition)
  • Business travel patterns

Project Scope

The analysis covered:

  • Data Cleaning: Processing 1,471 employee records totaling 22,065 data points
  • Visualization Creation: Developing clear visualizations to represent patterns in employee distribution, satisfaction, and attrition
  • Strategic Insight Delivery: Converting data patterns into actionable workforce insights

Technologies Used

Tableau Microsoft Excel HR Data Systems Git/GitHub

Skills Applied

Data Analysis Data Visualization Critical Thinking Data Storytelling Data Cleaning Interactive Design
